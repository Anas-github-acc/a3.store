
# Frontend for Distributed Redis-like KV Store

This is the frontend application for a distributed key-value store project inspired by DynamoDB and Cassandra. Built with React, Vite, TypeScript, Tailwind CSS, and shadcn-ui, it provides a modern UI to interact with the backend API and visualize the distributed system's architecture, node status, and monitoring data. Clone the repository, install dependencies, and run the development server to get started.
